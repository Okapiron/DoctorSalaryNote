# 医師給与ノート サポート

医師給与ノートをご利用いただきありがとうございます。

このページは、TestFlight配布およびApp Store提出時に利用するサポート案内です。

## サポート対象

以下の内容についてお問い合わせいただけます。

- アプリの起動や操作に関する不具合
- 勤務先、給与明細、書類、CSV出力に関する不具合
- Face ID / Touch IDロックに関する不具合
- 表示文言や使い勝手に関する改善提案
- プライバシーポリシーに関する質問

## お問い合わせ方法

GitHub公開後、GitHub Issuesをサポート窓口として利用する予定です。

Issueを作成する際は、可能な範囲で以下を記載してください。

- 利用している端末名
- iOSバージョン
- アプリのバージョンとビルド番号
- 発生した画面
- 再現手順
- 期待した動作
- 実際に起きたこと

給与額、勤務先名、源泉徴収票、給与明細画像などの個人情報・機密情報は、Issue本文やスクリーンショットに含めないでください。

## 注意事項

本アプリは、収入記録と書類整理の補助ツールです。

税務計算、確定申告書作成、税務助言、申告代行は行いません。税務判断が必要な場合は、税理士、税務署などの専門窓口へご確認ください。

## 緊急性の高い内容について

本アプリは医療行為、救急対応、税務申告の代行を目的としたものではありません。

緊急性の高い医療・税務・法律上の判断については、該当する専門機関へ直接ご相談ください。
