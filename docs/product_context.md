# 医師給与ノート product_context

## アプリ名

仮称：医師給与ノート

正式名称は今後変更する可能性がある。開発中は「医師給与ノート」として扱う。

## アプリの本質

医師のための収入台帳アプリ。

常勤、外勤、当直、スポット勤務、賞与などの給与・収入を記録し、年別・年度別に額面、手取り、控除、勤務先別収入、書類状況を見える化する。

## 背景

日本の医師は、常勤先に加えて、外勤、当直、日当直、スポット勤務など、複数の勤務先から収入を得ることが多い。

その結果、以下のような課題が生じやすい。

- どの勤務先からいくら支給されたか把握しにくい
- 額面年収と手取り年収を分けて管理しにくい
- 外勤収入や当直収入の年間合計がわかりにくい
- 年末調整や確定申告前に源泉徴収票、支払調書、給与明細を探すのが面倒
- 勤務先別、収入区分別の収入推移を振り返りにくい
- 紙やPDFの給与明細が散逸しやすい

このアプリは、これらを一元管理するためのローカル保存型iPhoneアプリとして設計する。

## 主なターゲットユーザー

主な対象は以下。

- 常勤先に加えて外勤をしている勤務医
- 当直、日当直、スポットバイトをしている医師
- 複数の勤務先から給与明細や源泉徴収票を受け取る医師
- 自分の額面年収、手取り年収、控除、外勤収入を把握したい医師
- 年末調整や確定申告前に書類を整理したい医師

## 提供価値

本アプリの主な価値は以下。

1. 医師の複数勤務先からの収入を一元管理できる
2. 額面、手取り、控除、手取り率を見える化できる
3. 年別、年度別に収入推移を確認できる
4. 勤務先別、収入区分別に収入を確認できる
5. 給与明細、源泉徴収票、支払調書などの書類を整理できる
6. 年末調整、確定申告前に不足書類を確認しやすい
7. ログイン不要・端末内保存で、給与情報を安心して管理できる

## アプリの位置づけ

本アプリは、税務申告ソフトではない。

本アプリは、給与明細、源泉徴収票、支払調書、収入記録を整理し、ユーザーが自分の収入状況と書類状況を把握するための補助ツールである。

## やらないこと

MVPでは以下を行わない。

- 確定申告書の作成
- 税額の正確な自動計算
- 税務助言
- 申告代行
- OCRによる完全自動入力
- マイナポータル連携
- 銀行口座連携
- クラウド同期
- サーバー保存
- 複数端末同期
- 勤務日や勤務対象期間の細かな管理

## 集計の基本思想

本アプリでは、年別と年度別の2種類の表示を扱う。

- 年別：1月〜12月。税務、書類管理、源泉徴収票、支払調書、CSV出力の基本軸。
- 年度別：4月〜翌年3月。主に医師の勤務年度感覚で収入推移を見るための補助表示。

年度別は、基本的にはホーム画面と分析画面における収入推移確認用として扱う。

書類管理は年別を基本とし、年度別切替は設けない。

## MVPの開発方針

最初から巨大な機能を作らず、段階的に実装する。

初回実装では、以下に絞る。

- SwiftUIアプリの土台
- SwiftDataによるローカル保存
- 下部タブ構成
- 勤務先管理
- 給与明細の登録、一覧、編集、削除

ホーム集計、書類管理、分析画面、Face ID、CSV出力、OCRは後続フェーズで実装する。

## セキュリティ方針

給与情報を扱うため、MVPでは以下を基本方針とする。

- ログイン不要
- 端末内ローカル保存
- 添付ファイルも端末内保存
- 外部サーバー送信なし
- クラウド同期なし
- Face ID / Touch IDロックは後続フェーズで対応
- プライバシーポリシーでは、給与情報を外部送信しない方針を明記する
