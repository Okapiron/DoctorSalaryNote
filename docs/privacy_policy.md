# 医師給与ノート プライバシーポリシー

最終更新日: 2026年5月29日

## 基本方針

医師給与ノートは、医師の給与・収入情報と関連書類を端末内で整理するためのアプリです。

給与情報、勤務先情報、源泉徴収票などはプライバシー性が高いため、本アプリはMVP時点ではログイン不要、端末内保存、外部サーバー送信なしを基本方針とします。

## 収集する情報

本アプリは、ユーザーが入力した以下の情報を端末内に保存します。

- 勤務先名、勤務先区分、メモ
- 給与明細・収入記録の支給年月、収入区分、額面、手取り、控除、メモ
- 書類情報の年、勤務先、書類種別、メモ
- ユーザーが選択したPDFまたは画像ファイル
- アプリ設定

## 保存場所

入力データと添付ファイルは、ユーザーの端末内に保存されます。

本アプリは、給与情報や添付ファイルを開発者のサーバー、外部サーバー、クラウドストレージへ自動送信しません。

## 外部送信

MVP時点の本アプリには、ユーザーの給与情報を外部サーバーへ送信する機能はありません。

CSV出力や共有は、ユーザーが明示的に操作した場合のみ行われます。共有先でのデータの扱いは、共有先サービスの規約と設定をご確認ください。

## 写真・ファイルの利用

画像の取込には、ユーザーが選択した画像のみを扱う仕組みを使用します。

PDFの取込では、ユーザーが選択したファイルのみをアプリ内に保存します。

本アプリは、写真ライブラリ全体や端末内ファイル全体を自動的に読み取ることはありません。

## Face ID / Touch ID

Face ID / Touch IDロックを有効にした場合、認証はAppleが提供する端末の認証機能を使用します。

本アプリは、顔や指紋などの生体情報を取得、保存、送信しません。

## データ削除

設定画面の全データ削除を実行すると、勤務先、給与明細、書類情報、保存済み添付ファイルを端末内から削除します。

この操作は元に戻せません。

## 税務・医療に関する注意

本アプリは、収入記録と書類整理の補助ツールです。

本アプリは、税務計算、確定申告書作成、税務助言、申告代行を行うものではありません。税務判断が必要な場合は、税理士、税務署などの専門窓口へご確認ください。
