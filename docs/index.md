# 医師給与ノート

医師給与ノートは、医師の給与・収入情報と関連書類を端末内で整理するためのiPhoneアプリです。

このページは、TestFlight配布およびApp Store提出時に利用する公開情報をまとめたものです。

## 公開ページ

- [サポート](support.md)
- [プライバシーポリシー](privacy_policy.md)

## アプリの位置づけ

本アプリは、収入記録と書類整理の補助ツールです。

税務計算、確定申告書作成、税務助言、申告代行は行いません。税務判断が必要な場合は、税理士、税務署などの専門窓口へご確認ください。

## プライバシー方針

MVP時点では、ログイン不要、端末内保存、外部サーバー送信なしを基本方針としています。

CSV出力や共有は、ユーザー操作によってのみ行われます。
